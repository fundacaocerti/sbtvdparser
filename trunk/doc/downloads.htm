Latest release (v0.31) 2010-11-29

New features:
&bull Demux - tool to select and filter PIDs for extraction into a new TS
&bull Crop - tool for selecting start and stop points in the progress bar and crop the TS into a new file
&bull Timestamps - when any item is selected in the PSI, caption or carousel trees the status bar shows the reception time-stamp of the item.
&bull Bitrates - the bitrate graphs now hold 5 min. of data and the sampling time selection was implemented.
&bull Save - now when saving a description text file every info. tree is saved along in it.
&bull Usability - las used folder is saved and recalled in the open/save dialogs and language selection is applied on-the-fly and persisted
Bugfixes:
&bull Many bugs related to timestamps
&bull Many internationalization tags were updated
Refatoring:
&bull Code now adherent to java 1.5
&bull Refactoring of the descriptor hierarchy + extended evt. descriptor
&bull Better performance with the usage of HashMaps is several points
